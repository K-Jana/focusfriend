========================
Instructions to Run
========================

1. Install dependacies in requirement.txt in Backend folder (In the command line go to the Backend folder and excute the following command)

pip install -r requirements.txt

2. Then execute the focusfriend.py flie with arugment server

python focusfriend.py server

3. Then open the Client\index.html by double clicking (in the default browser)

4. Enter your queries

5. Alternatively the focusfriend.py can be run as CLI mode

python focusfriend.py debug
